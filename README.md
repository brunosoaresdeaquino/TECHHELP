# TECHHELP - Suporte Técnico de Confiança
Visão Geral
TECHHELP é um serviço especializado em suporte técnico, oferecendo soluções de TI tanto para pessoas físicas quanto para empresas. Nossos serviços incluem suporte a hardware, software, redes e conectividade, com foco em manter a infraestrutura de TI de nossos clientes sempre funcional e eficiente.

Estrutura do Projeto
O projeto é composto pelos seguintes arquivos:

index.html: Estrutura da página principal e conteúdo da aplicação web.
style.css: Arquivo de estilos para a interface, garantindo uma aparência moderna e responsiva.
script.js: Script de funcionalidades interativas, como animações de rolagem e validação de formulários.
Imagens de suporte:
Hardware.jpg: Imagem representando suporte a hardware.
Suporte.jpg: Imagem representando suporte a software.
Conectividade.jpg: Imagem representando suporte a redes.
logo.jpg: Logotipo da empresa.
removebg.png e image.png: Imagens adicionais usadas na apresentação visual da página.
Funcionalidades
1. Navegação e Interface de Usuário
Header de Navegação: Um cabeçalho fixo com links de navegação que conduzem o usuário pelas seções principais da página.
Seção de Apresentação: Uma introdução destacando a missão da TECHHELP e seus serviços.
Seção de Depoimentos: Inclui feedbacks e uma chamada para ação para que os clientes entrem em contato para suporte imediato.
Seção de Áreas de Atuação: Mostra os serviços oferecidos com cartões clicáveis.
Formulário de Contato: Para envio de mensagens rápidas e solicitação de suporte.
2. Interatividade e Funcionalidades JS
Rolagem Suave: Implementada para uma navegação mais amigável entre as seções​(script).
Animação de Entrada: Elementos aparecem suavemente ao serem visualizados​(script).
Validação de Formulário: Verifica se todos os campos obrigatórios foram preenchidos antes do envio, destacando-os em vermelho caso contrário​(script).
3. Estilização Personalizada
O arquivo style.css contém:

Estilos para o layout geral, incluindo o cabeçalho, rodapé, e seções de conteúdo​(style).
Animações e transições para efeitos de foco e hover, melhorando a acessibilidade​(style).
Responsividade para dispositivos móveis através de media queries​(style).
4. Recursos de Imagem
As imagens fornecidas são usadas para representar graficamente os diferentes serviços oferecidos e para embelezar a interface​(index).

Como Usar
Abra o arquivo index.html em um navegador para visualizar a página completa.
Personalize as imagens e conteúdo no arquivo index.html conforme necessário.
Modifique o arquivo style.css se desejar ajustar o design ou adicionar novos estilos.
Adapte o script script.js para incluir funcionalidades adicionais ou ajustes nos comportamentos interativos.
